a tag

    a tag 속성에는 여러가지가 있다.

  • href : 속성값에는 주소를 넣음

      • 메일 관련

    • mailto:{메일주소} : 메일 보내기
    • mailto:{메일주소}?subject=메일제목 : 메일 제목 미리 입력
    • mailto:{메일주소}?body=내용본문 : 메일 내용 미리 입력
    • 내 주소 호호
    • 내 주소 교수님 죄송합니다..
    • 내 주소 교수님 A+좀 줘요
    • 내 주소 교수님을 위한 편지

      • 전화 관련

    • tel:{전화번호} : 전화걸기
    • tel:{010-6607-4148} : 이것도 가능
    • tel:+국가번호{국가번호} : 국제전번도 가능

      • 문자 관련

    • sms:{전화번호} : 문자 보내기
    • sms:{전화번호}?body=문자 내용 : 문제 내용까지
  • target : 링크를 어디에서 열지 정함
    • _self : 기본값으로 현재 페이지에서 링크를 처리
    • _blank : 새 창 또는 새 탭에서 링크를 처리
    • _parent : 부모 창에서 연다 -> 창 이동(없으면 _self 처리)
    • _top : 전체 브라우저 창에서 가장 상위 창에서 연다
  • download : 링크 대상을 다운로드함
  • download={파일명} : 링크 대상의 이름을 바꾸고함 다운로드함
  • title : 링크에 마우스 커서를 오릴ㄹ 떄 도움말 설명 설정 -> 이미지에서 사용

이거라벨

tag 는 열림 태그와 닫힘 태그로 이루어져 있다

하지만 닫힘 태그가 없는 태그를 보고 셀프 클로징 태그라고 함

아래와 같이 이미지에 링크를 달 수 있다.

img tag는 셀프 클로징 태그이다.

img tag는 몇가지 속성이 있다.

  • src : 이미지의 경로
  • alt : 대체 텍스트(이미지를 표시할 수 없을 때 출력할 내용)

    ++ alt 속성이 없는 것은 검색 앤진이 싫어함 so alt=""이라도 하는게 좋다.

  • width : 이미지의 가로 크기
  • height : 이미지의 세로 크기
  • loading : 이미지의 로딩 방식

++ img tag 추가 심화

  • if width와 height 정하지 않으면

    • 이미지 원본 크기 그대로 나옴

  • width 속성으로 가로만 정하면 세로 크기는 원본의 가로 세로 비율에 맞춰짐
  • height 속성으로 세로만 정하면 가로 크기는 원본의 세로 가로 비율에 맞춰짐
  • 가로와 세로 크기를 원본에 상관없이 가로, 세로 크기로 정해짐

  • loading 속성
    • eager : 바로 이미지 로딩(default value)
    • lazy : 이미지 뷰포트의 일정거리 안으로 들어오면 불러옴

      • but IE에서 loading 속성을 지원 안함 조심

추가적으로 alt 속성이면 시맨트 태그임
