form tag

    form tag 속성

  • name : 폼을 식별하기위한 이름
  • action : 폼을 전송할 서버 쪽 스크립트 파일을 지정
  • accept-charset : 폼 전송을 사용할 문자 인코딩을 지정
  • target : 어디서 창을 열껀지 지정
  • method : 폼을 서버에 전송할 http 메소드를 지정(GET or POST)
    • GET 방식 : 폼 데이터를 가져와 서버에 보낼 때, 폼 데이터를 URL 끝에 붙여 전송함
      ex : http://localhost8080/form.jsp?name=jun&address=suwon
    • POST 방식 : 폼 데이터를 가져와 서버에 보낼 때, 내부적으로 보이지않게 전송
      ex : http://localhost8080/form.jsp

      • 정리 : GET 방식은 지정된 리소스에서 자원을 읽을 경우에 GET방식 사용
      나머지 방식에는 POST방식을 사용 WHY? GET방식은 데이터가 외부에 노출되어 보안에 취약함

목록 tag

select : 항목을 선택할 수 있는 태그

    select의 속성

  • size : 한번에 표시할 수를 의미
  • multiple : 다중선택을 허용할것인지
아래 태그는 select의 하위 tag
  • optgroup tag: select tag 내부에서 목록들을 그룹화할 경우에 사용
    - label 속성을 사용하여 그룹 이름을 지정
  • option tag : 목록들을 말함
    - optgroup tag 하위 tag로 사용하기도 함
